Wild Sleep Navigator

Aplikace používá AI pro analýzu míst k přespání dodávkou. Potřebuje tvůj Anthropic API klíč — uloží se jen do tohoto prohlížeče (localStorage), nikam se neposílá.

Klíč získáš na console.anthropic.com → API Keys → Create Key.
Bez klíče aplikace nefunguje — API volání se platí dle spotřeby (~$0.003 / analýza).
